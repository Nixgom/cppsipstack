==============
SipSpeedLinux
==============

1. 개요

 * C++ SIP stack / SipUserAgent 를 이용한 SIP 성능 테스트 프로그램 개발
 
 * 목적
   - C++ SIP User Agent 를 이용하여서 SIP 서버 성능 측정 프로그램을 개발한다.
   - 리눅스 및 윈도우에서 모두 사용 가능한 프로그램을 개발한다.

 * 주요 기능
   - CPS 측정
